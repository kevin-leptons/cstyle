\section{gitignore file}

\begin{crules}
    \cmust{Put "tmp/", "dest/", "dist/" into .gitignore}
    \cmust{Put "*.swp/", "*.log" into .gitignore}
    \cnot{Put source code files into .gitignore}
\end{crules}

In simple, put all of files aren't source code into ".gitignore" file.
